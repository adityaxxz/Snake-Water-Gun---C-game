# Snake-Water-Gun---C-game

A simple command-line based Snake Water Gun game implemented in C language.

How to play -->     

1)Compile the game using a C compiler such as GCC or Run the .exe file in windows power shell.
2)You will be prompted to enter your choice (s for snake, w for water, or g for gun).
3)The computer will also make a choice randomly.

The winner will be determined based on the following rules:
    1.Snake beats gun (snake drinks water and gun rusts)
    2.Water beats snake (water drowns snake)
    3.Gun beats water (gun shoots snake)

The game will continue until you choose to exit.
